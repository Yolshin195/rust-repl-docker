# rust-repl-docker
Rust REPL in Docker/Podman - Этот проект предоставляет готовый к использованию образ Docker/Podman с предустановленным evcxr_repl. Он идеально подходит для экспериментов с кодом на Rust, быстрого прототипирования или обучения, не засоряя вашу основную операционную систему установкой различных версий Rust и системных библиотек.
